StockPilot - 个人 AI 投研助理StockPilot 是一个基于 Python 的工具，旨在自动化收集股票市场数据，计算技术指标，并生成结构化的提示词（Prompt），供大语言模型（LLM）为您提供投资建议。功能特性 (Features)自动化数据获取: 使用 AkShare 接口拉取股票、ETF 和指数的实时/历史数据。技术分析: 自动计算 KDJ、MACD、BOLL (布林带) 和移动平均线 (MA)。持仓管理: 跟踪您的特定持仓，并计算当前的盈亏比例。环境感知: 支持获取主力资金流向和新闻头条（在配置中可选），为 AI 提供更多上下文。Prompt 生成: 输出结构化的 JSON 提示词，可直接发送给 DeepSeek、ChatGPT 或 Claude 进行分析。安装与配置 (Setup)安装依赖:pip install -r requirements.txt
配置:系统使用您上传的 config.yaml 文件。请确保文件中的股票代码和持仓信息是最新的。股票代码: 必须符合 600519.SH 或 300750.SZ 的格式。分组 (Groups): 将股票标记为 "当前持仓"，脚本会自动触发持仓分析逻辑。运行:python stock_pilot.py
输出使用 (Output)脚本运行后，会在控制台输出一大段文本。复制 === 已生成 Prompt === 下方的文本。粘贴 到您喜欢的大模型对话框中（例如 DeepSeek Chat）。该内容也会自动保存到本地文件 last_generated_prompt.md，方便查看。自定义配置 (Customization)添加股票: 编辑 config.yaml 中的 security_pool 列表。修改指标: 修改 stock_pilot.py 中的 IndicatorCalculator 类。开启新闻/资金流: 在 config.yaml 中将 fetch_main_money_flow 或 fetch_news_sentiment 设置为 true。
